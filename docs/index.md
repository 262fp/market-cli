# Fundamentos de Programacion

**Recursos del curso 2026**

**Institucion:** Universidad Peruana Union  
**Ano:** 2026

---

## Presentacion

Este espacio fue ajustado para el curso de Fundamentos de Programacion. El eje del curso es la construccion progresiva de una aplicacion de consola para la gestion de una entidad de negocio mediante programacion estructurada.

## Producto del curso

**Aplicacion de consola para la gestion de una entidad de negocio (Producto, Cliente, Usuario u otra equivalente), desarrollada mediante programacion estructurada, integrando estructuras secuenciales, condicionales, repetitivas, subprogramas, recursividad, estructuras de datos basicas y archivos para registrar, consultar, actualizar, eliminar, procesar y presentar informacion.**

## Resultado de aprendizaje del curso

Desarrolla una aplicacion de consola para la gestion de una entidad de negocio, aplicando programacion estructurada, estructuras de control, subprogramas, recursividad, estructuras de datos basicas y archivos para procesar y presentar informacion.

## Unidades

### Unidad 1

- Resultado: aplica estructuras secuenciales y condicionales para resolver problemas simples y construir un menu basico para la gestion de una entidad.
- Producto: menu basico de gestion de una entidad.

### Unidad 2

- Resultado: implementa estructuras repetitivas, subprogramas, recursividad y estructuras de datos basicas para desarrollar una aplicacion de consola con operaciones CRUD en memoria y procesamiento de informacion.
- Producto: aplicacion de consola con CRUD en memoria y procesamiento de datos.

### Unidad 3

- Resultado: integra persistencia de datos y generacion de salidas en la aplicacion desarrollada durante el curso.
- Producto: aplicacion de consola completa para la gestion de una entidad.
